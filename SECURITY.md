# Security Policy (Draft)

## Reporting
If you discover a security issue, report it privately to the project owner/admin team.

## Principles
- Least privilege (RBAC)
- Sensitive links (Teams join URLs, subscription tokens) are not logged broadly
- Rate limiting for auth and sign-up endpoints
- Audit log for admin actions

## Data Protection
- Follow POPIA-aligned practices (data minimization, purpose limitation, retention)
- Operational baseline reference: `docs/privacy-baseline.md`
